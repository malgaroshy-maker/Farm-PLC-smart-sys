# HARDWARE_BOM_DRAFT.md

## 1. Purpose

This is a planning BOM, not a final procurement list.

## 2. Core Control Cabinet Blocks

- Siemens-oriented PLC CPU
- required digital input modules
- required digital output modules
- analog input modules
- power supply
- surge and circuit protection
- relays/contactors or interface relays as needed
- terminal blocks
- cabinet HMI optional
- industrial router or gateway
- GSM/SMS module or compatible gateway
- UPS optional for control electronics

## 3. Field Instrumentation

### Mandatory for smart phase 1
- electric actuators for each field valve
- one mainline pressure transmitter
- one mainline flow meter
- one tank level sensor/transmitter
- one soil moisture sensor per zone
- pump run/fault feedback wiring

### Strongly recommended soon after
- well level or dry-run protection
- per-zone pressure or flow where economically justified
- power quality or energy metering for pumps

## 4. Mechanical / Hydraulic Items Likely Needed

- valve retrofitting or replacement
- actuator-compatible valve hardware
- instrument tees/tappings
- signal cables and conduit
- enclosure/junction boxes
- mounting accessories

## 5. Site Survey Data Still Needed

- exact number of zones
- valve sizes and types
- pipe diameters
- pump HP/kW, head, flow
- distance to each field
- available panel location
- telecom signal quality
